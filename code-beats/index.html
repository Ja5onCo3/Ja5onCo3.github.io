// How to Use Code Beats

Getting Started

Type commands in the left panel and press ▶ Run to execute. Each line is one command. Lines starting with // are comments and are ignored. Load a preset from the bar below the editor to hear something instantly.

Synth & Melody

play(C4, 0.5)Play note C4 for 0.5 seconds. Notes: A–G, sharps (#) and flats (b), octave 1–8
synth(sine)Set waveform: sine | square | triangle | sawtooth
bpm(120)Set tempo in beats per minute (40–240)
volume(0.8)Master volume (0.0–1.0)
melody_loop(true)Loop play() notes in sync with drum sequencer (true | false)

Envelope (ADSR)

Shape how notes start, sustain, and fade. Changes are shown live in the ADSR curve diagram.

attack(0.02)Time to reach full volume (0.001–2.0 s)
decay(0.1)Time to drop to sustain level (0.001–2.0 s)
sustain(0.5)Volume held while note is active (0.0–1.0)
release(0.8)Time to fade after note ends (0.001–4.0 s)

Effects

reverb(0.6)Room reverb wet mix (0.0–1.0)
delay(0.3)Echo feedback delay wet mix (0.0–1.0)

Drum Sequencer

Pass an 8-step array of 1s and 0s. Combine with loop(true) to start the sequencer. You can also click cells directly on the grid.

kick([1,0,0,0,1,0,0,0])Kick drum pattern
snare([0,0,1,0,0,0,1,0])Snare pattern
hihat([1,1,1,1,1,1,1,1])Hi-hat pattern
loop(true)Start looping (true | false)

Bass Line

Program an independent bass synth with its own waveform, envelope, and effects. Bass runs in sync with the drum sequencer — loop(true) must be active.

bass([C2,0,G1,0,C2,0,Bb1,0])8-step bass pattern. Use note names or 0 for rest
bass_synth(sawtooth)Bass waveform: sine | square | triangle | sawtooth
bass_vol(0.7)Bass volume (0.0–1.0)
bass_attack(0.05)Bass attack (0.001–2.0 s)
bass_decay(0.2)Bass decay (0.001–2.0 s)
bass_sustain(0.6)Bass sustain level (0.0–1.0)
bass_release(0.4)Bass release (0.001–4.0 s)
bass_reverb(0.2)Bass reverb wet mix (0.0–1.0)
bass_delay(0.1)Bass delay wet mix (0.0–1.0)

Exotic Percussion

Program exotic drums from the 🥁 Percussion drawer. Each drum has an independent 8-step pattern running in sync with the main sequencer. Open the drawer to toggle cells or use code commands.

djembe([1,0,0,1,0,0,1,0])8-step pattern for any exotic drum
tabla([1,0,1,0,0,0,0,0])Same format for: tabla, darbuka, talking, dunun, cajon, conga, taiko, dhol, framedrum
perc_vol(djembe, 0.8)Set volume for a specific drum (0.0–1.0)

Melodic Instruments

Select and play real instruments from the 🎵 Melodic drawer. Sampled instruments load on first use. Use dedicated commands or route play() through a selected instrument.

piano(C4, 0.5)Play a note with a specific instrument. Available: piano, electricpiano, organ, warmpad, atmosphere, clavicylinder, pluckedbass, electriccello, violin, guitaracoustic
instrument(piano)Set active instrument — all play() commands route through it
inst_vol(piano, 0.8)Set instrument volume (0.0–1.0)
melody_loop(true)Loop play() notes in sync with drum sequencer (true | false)

Tips

Commands execute top to bottom. Set synth, bpm, and envelope before play() for best results. The ↺ Loop button toggles the drum sequencer without re-running code. Stop clears all playback. Sampled instruments must finish loading before they play — a loading indicator appears in the Melodic drawer. Exotic percussion requires at least one drum pattern active and loop(true) to run.
